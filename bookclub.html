ABOUT

It's A Hardback Life Bookclub is a bookclub application that has client and admin logins, the ability to search and save books, and save book reviews. It utilizes the OpenLibrary API and allows users to chat about their favorite books in real time.

TECH  


Technologies used include:
 JavaScript
 CSS & Bulma
 Handlebars
 Figma
 OpenLibrary API
 Pusher API
 MySQL
 Node.js & Express.js
